# Loan Approval Calculator

A Python application that simulates real-world loan approval decisions based on user financial input.

## Features
- Calculates monthly loan payments
- Determines loan approval or denial
- Uses conditional logic and input validation
- Handles user input for financial data

## Technologies
- Python

## Purpose
This project demonstrates the use of programming logic to solve real-world financial decision-making problems.

## How to Run
1. Download the project files
2. Open in a Python environment
3. Run the main Python file
